# Cancel Order `DELETE` https://api.dhan.co/v2/orders/{order-id} Cancel a pending order. > **NOTICE:** Order Placement, Modification and Cancellation APIs requires Static IP whitelisting - [refer here](/api/v2/guides/authentication#setup-static-ip) ### Path Parameters | Parameter | Type | Required | Description | |-----------|------|----------|-------------| | order-id | string | Yes | Order ID of the Order being cancelled | ### Response Fields | Field | Type | Required | Description | |-------|------|----------|-------------| | orderId | string | No | Order specific identification generated by Dhan | | orderStatus | enum string | No | Last updated status of the order Values: `TRANSIT`, `PENDING`, `REJECTED`, `CANCELLED`, `TRADED`. | ### Status Codes | Code | Description | |------|-------------| | 200 | Order cancelled | ### Example Request ```bash curl -X DELETE https://api.dhan.co/v2/orders/{order-id} \ -H "access-token: " ```
