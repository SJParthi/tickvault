# Historical Data Historical Data This API gives you historical candle data for the desired scrip across segments & exchange. This data is presented in the form of a candle and gives you timestamp, open, high, low, close & volume. | Method | Endpoint | Description | |--------|----------|-------------| | POST | `/charts/historical` | Get OHLC for daily timeframe | | POST | `/charts/intraday` | Get OHLC for minute timeframe |
