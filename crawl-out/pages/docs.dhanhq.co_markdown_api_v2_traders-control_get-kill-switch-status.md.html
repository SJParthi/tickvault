# Kill Switch Status `GET` https://api.dhan.co/v2/killswitch Check whether kill switch is active for the current trading day. The API allows you to check kill switch status for your account - whether it is active for the current trade or not. ### Response Fields | Field | Type | Required | Description | |-------|------|----------|-------------| | dhanClientId | string | No | User specific identification generated by Dhan | | killSwitchStatus | string | No | Status of Kill Switch Values: `ACTIVATE`, `DEACTIVATE`. | ### Status Codes | Code | Description | |------|-------------| | 200 | Current kill switch status | | 401 | Unauthorized | ### Example Request ```bash curl -X GET https://api.dhan.co/v2/killswitch \ -H "access-token: " ```
