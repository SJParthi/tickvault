# Get Daily Historical Data `POST` https://api.dhan.co/v2/charts/historical Get daily OHLC candle data for backtesting and analysis. Retrieve OHLC & Volume of daily candle for desired instrument. The data for any scrip is available back upto the date of its inception. ### Request Body Parameters | Parameter | Type | Required | Description | |-----------|------|----------|-------------| | securityId | string | No | Exchange standard ID for each scrip. refer here | | exchangeSegment | enum | No | Exchange & segment (see Annexure) Values: `NSE_EQ`, `NSE_FNO`, `BSE_EQ`, `BSE_FNO`, `MCX_COMM`, `IDX_I`. | | instrument | string | No | Instrument type of the scrip (see Annexure) | | expiryCode | int | No | Expiry of the instrument (for derivatives) | | oi | string | No | Open Interest data for F&O: true or false | | fromDate | string | No | Start date (YYYY-MM-DD) | | toDate | string | No | End date (YYYY-MM-DD, non-inclusive) | ### Response Fields | Field | Type | Required | Description | |-------|------|----------|-------------| | open | float | No | Open price of the timeframe | | high | float | No | High price in the timeframe | | low | float | No | Low price in the timeframe | | close | float | No | Close price of the timeframe | | volume | int | No | Volume traded in the timeframe | | open_interest | int | No | Open interest (for F&O instruments) | | timestamp | int | No | Epoch timestamp | ### Status Codes | Code | Description | |------|-------------| | 200 | Historical OHLC data | ### Example Request ```bash curl -X POST https://api.dhan.co/v2/charts/historical \ -H "access-token: " \ -H "Content-Type: application/json" ```
