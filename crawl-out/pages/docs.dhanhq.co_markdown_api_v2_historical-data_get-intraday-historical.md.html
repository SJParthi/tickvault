# Get Intraday Historical Data `POST` https://api.dhan.co/v2/charts/intraday Get intraday OHLC candle data (1, 5, 15, 30, 60 minute intervals). Retrieve Open, High, Low, Close, OI & Volume of 1, 5, 15, 30 and 60 min candle for desired instrument for last 5 years. This data available for all exchanges and segments for all active instruments. ### Request Body Parameters | Parameter | Type | Required | Description | |-----------|------|----------|-------------| | securityId | string | No | Exchange standard ID for each scrip. refer here | | exchangeSegment | enum | No | Exchange & segment (see Annexure) Values: `NSE_EQ`, `NSE_FNO`, `BSE_EQ`, `BSE_FNO`, `MCX_COMM`, `IDX_I`. | | instrument | string | No | Instrument type of the scrip (see Annexure) | | interval | enum | No | Minute intervals Values: `1`, `5`, `15`, `30`, `60`. Default: `5`. | | oi | string | No | Open Interest data for F&O: true or false | | fromDate | string | No | Start date (YYYY-MM-DD) | | toDate | string | No | End date (YYYY-MM-DD) | ### Response Fields | Field | Type | Required | Description | |-------|------|----------|-------------| | open | float | No | Open price of the timeframe | | high | float | No | High price in the timeframe | | low | float | No | Low price in the timeframe | | close | float | No | Close price of the timeframe | | volume | int | No | Volume traded in the timeframe | | open_interest | int | No | Open interest (for F&O instruments) | | timestamp | int | No | Epoch timestamp | ### Status Codes | Code | Description | |------|-------------| | 200 | Intraday OHLC data | ### Example Request ```bash curl -X POST https://api.dhan.co/v2/charts/intraday \ -H "access-token: " \ -H "Content-Type: application/json" ```
