### We offer following types of Data through our APIs

##### Realtime Data API

Realtime data updating at 1 second frequency is available for NSE Stocks, NSE Indices, NSE F&O, NSE Currency ,MCX,BSE Stocks, BSE Indices & BSE F&O. This is L1 data with single best Bid & Ask details.

##### Realtime OptionChain API

If you are an OptionChain trader, we also have special API for OptionChain data. Here, you can get realtime price of entire OptionChain of the underlying (say NIFTY, BANKNIFTY, RELIANCE, etc.) in single call.

##### Realtime Exchange Snapshot API

If you do not need realtime data every second but if you need snapshot data of entire Exchange after every few minutes, we also provide API for the same purpose. Currently supported periods are Minute (1,2,5,10,15,30), Hour and Day (end-of-day)

## Type Of APIs Available

> Source: https://globaldatafeeds.in/global-datafeeds-apis/global-datafeeds-apis/introduction/type-of-apis-available/ — last updated 2024-07-09T18:38:08 — captured 2026-07-15

### Type of APIs Available

- **WebSockets :** Language neutral. Can be used for any programming language and for any Operating System (windows / Linux). This is currently the go-to choice of most web, mobile and desktop app developers for developing applications in Python, Java, JavScript, NodeJS, CSharp and any other language which supports WebSockets. This is "Push Type API" – meaning – once connected, server will push you realtime and snapshot data of subscribed symbols whenever new update is available. Due to its popularity, we also have 3rd party client available for the same
- **DotNet :** Can be used in any DotNet language like C#, VB.NET. This is a "Push Type API" – meaning – once connected, server will push you realtime and snapshot data of subscribed symbols whenever new update is available.
- **COM :** Can be used in languages like C, C++ and any language that support COM. This is a "Push Type API" – meaning – once connected, server will push you realtime and snapshot data of subscribed symbols whenever new update is available.
- **RESTful :** Language neutral. Like WebSockets API, this can be used for any programming language and for any Operating System (windows /Linux). This is simplest form of API with lowest implementation periods. This is "Pull Type API" – meaning – you will need to send request "everytime" when you need latest data.

## Symbol Availability

> Source: https://globaldatafeeds.in/global-datafeeds-apis/global-datafeeds-apis/introduction/symbol-availability/ — last updated 2024-07-10T10:58:27 — captured 2026-07-15

### Symbol Availability

- Data of all traded symbols from all supported exchanges is available. However, as per package purchased by you, concurrent access of from 50 to 500 symbols will be made available to you.
- Irrespective of whichever package you choose, you can request data of any symbol at any time. For example, there are around 2000 symbols under NSE Stocks. If you have purchased 50 symbols package and getting realtime updates from 50 symbols, you may send request to our server to stop data for any stocks and replace them with new stocks. This change happens in realtime without any manual intervention (Subscribe & Unsubscribe mechanism)
