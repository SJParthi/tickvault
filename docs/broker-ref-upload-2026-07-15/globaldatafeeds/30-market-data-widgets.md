### Widget catalog

Each widget links to the widgets portal at **https://webwidgets.globaldatafeeds.in/** ("Get Widget"), where widgets are configured and obtained. The product page itself demonstrates each widget with an animated GIF preview; it does not include inline embed code snippets — embedding is handled via the widgets portal.

| Widget | Description |
| --- | --- |
| **Top Gainers / Losers** | Get a quick glance at top gainers and losers on Indian exchanges. Updates every few minutes. Analyze real-time price action, volume, and performance. |
| **Advance Chart** | A powerful charting solution you can easily embed on any website fully customizable to match your brand. |
| **Watchlist** | Track real-time index data with live charts and performance metrics, all in one seamless widget. |
| **Symbol Short** | Get instant access to key symbol data, including quotes and financials, in a clear snapshot. |
| **Symbol Detailed** | Track comprehensive symbol data with live price charts, trends, 52 week values all in one widget. |
| **Ticker Tape** | Track real-time market activity with our dynamic ticker tape, always up to date. |
| **Market Data** | Historical Market Data at your finger tips. Includes Open, High, Low, Close, Volume, Open Interest, and Date. |
