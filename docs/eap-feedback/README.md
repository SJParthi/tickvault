# Claude Code Early Access Program — Feedback

This directory collects feedback for the **Claude Code Early Access Program (EAP)**.

## Contents

- **`tickvault-eap-dossier.md`** — the TickVault dossier. Field-collected feedback from
  real Claude Code sessions on this codebase, organized by the EAP team's 5 feedback
  topics (use-case fit, reliability, memory, proactivity, session-UX). This is a
  **living / append-only** document: new findings are appended under the relevant topic
  over time; existing entries are not rewritten.

## Plan

A separate **BruteX dossier** will be added to this directory. Once both dossiers are
here, the combined set will be submitted **together** in a single email to
**claude-code-early-access@anthropic.com**.

This branch/directory is the shared collection point for that consolidation. There is no
rush to merge — it is a holding / collaboration location for the EAP feedback until the
combined submission is ready.
