## Notes for Claude Code usage

- Dhan v2 feed is **tick-by-tick event-based** (contrast: Groww is ~1/sec snapshot-based) — limits: **5 WebSocket connections × 5000 instruments = 25,000 total**, max 100 instruments per subscribe message.
- All WS responses are **binary little-endian**; request messages are JSON.
- `access-token` header on api.dhan.co; token validity 24h (extendable via consent flow / renew).
- Sandbox base URL and behaviour documented in `01-overview-getting-started.md`.
